# Albert-Carlos-Jeronimo

**Ideas principales:**


1. Sonido skate en MACBA/Plaza universitat, clima y turismo. 
   
   
2. Relación entre condiciones climáticas y actividad skate / sonido en MACBA
3. Una instalación donde el sonido real modifique un modelo 3D de MACBA 
4. Movimientos y trucos de skate en formas paramétricas generativas, transforma cuerpo en arquitectura digital.
   
 **Posibles formas de presentación:** 
- creación de visuales en processing a partir de data + Proyección (videomapping)
- Creación modelo 3D en grasshopper a partir de audio + Proyección  (videomapping)
- Audio/data skate + generación de imágen/animación + Proyección (videomapping)
